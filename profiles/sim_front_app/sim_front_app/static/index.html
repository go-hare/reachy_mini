Flow

  1. ReachyMiniApp 常驻托管当前 app 的 runtime
  2. 页面通过 /ws/agent 把文本或浏览器语音生命周期事件发进 runtime
  3. front 先回一轮可见 hint，再把消息投给 kernel
  4. kernel 处理完成后，再交给 front 输出 final
  5. 页面持续监听 surface_state、hint、final 和错误事件
